Measurements from Harvard Forest and Chacaltaya can be accessed from existing repositories at the following DOI's:

Harvard Forest - Obrist et al. (2021) - https://doi.org/10.1073/pnas.2105477118

Chacaltaya (Magand and Dommergue, 2023)
Link: https://doi.org/10.25326/347
Key files to download to reproduce analysis in this paper are L1_TGM_CHC_2014.csv and L1_TGM_CHC_2015.csv

Once downloaded to this repository, data processing and plotting scripts will be able to run.
